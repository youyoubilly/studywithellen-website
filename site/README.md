# Ellen Wang English Studio — Site

Astro static landing page for Ellen Wang English Studio.

## Quick start

```bash
npm install
npm run dev       # http://127.0.0.1:4321
npm test          # Vitest
npm run test:e2e  # Playwright (mobile / tablet / desktop)
npm run build     # → dist/
```

## Public URL (target)

`http://hk1.youyoubilly.com:8080`
