# 视觉与 UX 设计建议

## 品牌气质

- **专业** — 像学术英语教练，不是培训机构广告
- **成熟** — 服务高中生与成人，避免卡通、积木、彩虹幼教风
- **可信** — 证书、机构名、履历时间线清晰可见
- **简洁** — 留白多、信息分层清楚；家长微信里 30 秒能看懂

## 参考调性（非抄版）

- 大学语言中心官网的克制感
- 高端私教个人品牌站（单人工作室）
- 避免：少儿英语机构的鲜艳插画风

## 配色建议

| 角色 | 色值建议 | 说明 |
|------|----------|------|
| 主色 Primary | `#1E3A5F` 深海蓝 | 信任、学术 |
| 辅色 Accent | `#C9A227` 或 `#B8860B` 暗金 | CTA 点缀，不大面积 |
| 背景 Background | `#FAFAF8` 暖白 | 比纯白柔和 |
| 文字 Text | `#1A1A1A` / `#4A4A4A` | 正文二级灰 |
| Employers 区块底 | `#F0F4F8` 浅蓝灰 | 与主内容微区分 |

## 字体

**中文：** 系统栈 `PingFang SC`, `Microsoft YaHei`, `sans-serif`  
**英文：** `Georgia` 或 `Merriweather`（标题）+ `system-ui`（正文）  
不必强求 Web Font，微信内加载快更重要。

## 摄影与图片

| 素材 | 规格建议 | 说明 |
|------|----------|------|
| Hero 头像 | 3:4 或 1:1，≥800px | 专业半身照，浅背景 |
| OG 分享图 | 1200×630 | 左文右图或居中头像+标语 |
| 微信二维码 | ≥430×430 PNG | 工作号 Ellen_inGZ |
| 证书照 | 可选横排小图 | CELTA / DELTA / CertIBET |

## 组件建议

**顶栏：** 左 Ellen Wang · 右「微信咨询」按钮  
**Hero：** 大标题 + 一行背书 + 两按钮 + 右侧/下方头像  
**服务卡片：** 三列（桌面）/ 单列（手机）；每卡：图标或序号 + 标题 + 3 条 bullet  
**履历：** 竖向时间线或简洁表格  
**FAQ：** 手风琴折叠，默认展开前 2 条  
**Contact：** 二维码居中放大，邮箱在下

## 动效

- 第一版：**无** 或仅 hover 按钮变色
- 禁止：全屏 parallax、自动轮播、弹窗

## 无障碍 & 可读性

- 正文字号手机端 ≥ 16px
- 行高 1.6–1.8
- 按钮文字对比度 WCAG AA
